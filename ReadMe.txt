	  3. בשורות מתחתיה שנו בשורות האלו רק את המילים active_block ו passive_block לפי הסדר שתרצו.

	  4. לאחר מכן שמרו את הקובץ (ctrl+s).
	  
- בסוף הניסוי:
	- ייפתח באופן אוטומטי חלון הורדה של קובץ התוצאות.
	- אם מסיבה כלשהי החלון לא נפתח אז יש אפשרות להוריד את הקובץ ע"י לחיצה על הכפתור Download CSV.
	- אין לשנות את שם הקובץ מכיוון שבתוך שם הקובץ יש מספר המייצג "חותמת זמן" כדי שלא יהיו שני קבצי תוצאה עם אותו שם.
	- יש לשמור את כל קבצי התוצאות בתיקיה אחת.
	- למיזוג כל קבצי התוצאות יש להעביר את הקובץ merge results.exe לתוך התיקיה שמכילה את קבצי התוצאות ולהפעיל אותו.
	- לאחר הפעלת המיזוג יווצר קובץ חדש בשם result-merged.csv עם תוצאות כל הניסויים שבתיקיה.
	- אין למחוק את קבצי תוצאות הניסויים לאחר המיזוג מכיוון שמיזוג חדש דורס את קובץ המיזוג הישן.
